Sketches

These pictures represent a visual depiction of the Visually Impaired Support App, or VISA. The app screens allow for voice navigation and feedback for visually impaired users. Each screen provides a different capability for the user. The user can get a weather forecast, get walking directions to a location, search the web or place an online order for food.
